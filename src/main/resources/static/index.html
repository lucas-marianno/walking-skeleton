API Documentation

Hello endpoint

A simple hello world endpoint for testing connectivity.

  • Endpoint (live): http://skeleton.olaria.online/hello?name=example
  • Endpoint: GET /hello
  • Returns: 200 OK + body hello [name]
  • Params:
    • name: replaces "stupid" with the provided name
    • delay: delays the response by "n" seconds

Grafana endpoint

Provides the entrypoint to the Grafana admin interface.

Grafana's provisioning is already set up. The data source is connected to Prometheus and a dashboard has been pre-created.

  • Endpoint (live): http://skeleton.olaria.online/grafana | Live dashboard here.
  • Endpoint (remote): GET /grafana
  • Endpoint (local): GET localhost:3000
